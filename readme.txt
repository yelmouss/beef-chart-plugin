== Description ==

Yelmouss French Beef Cuts Chart is a comprehensive WordPress plugin that provides an interactive visualization of French beef cuts with real-time pricing management. Perfect for butchers and meat shops looking to enhance their online presence.

**Features:**
- Interactive SVG map with 29 French beef cuts
- Real-time price management system
- Admin interface for easy management
- Shortcode and Gutenberg block support
- Responsive design for all devices
- Security hardened with prepared statements
- Multi-language ready (i18n support)
- Role-based access (editors and administrators)

**Perfect for:**
- Butcher shops and meat markets
- Food blogs and culinary websites
- Restaurant websites
- Agricultural and farming websites
- Food industry professionals

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/beef-chart` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Beef Chart menu to configure your beef cuts and prices
4. Use the shortcode `[beef_chart]` to display the interactive chart on any page or post

== Usage ==

**Shortcode Usage:**
```
[beef_chart]
[beef_chart width="800px" height="500px"]
[beef_chart width="100%" height="600px"]
```

**Gutenberg Block:**
1. Add a new block in the Gutenberg editor
2. Search for "Beef Chart"
3. Configure width and height as needed

**Admin Management:**
1. Go to **Beef Chart** in your WordPress admin menu
2. Update prices and availability for each beef cut
3. Preview changes in real-time
4. Save your modifications

== Frequently Asked Questions ==

= How do I display the beef chart? =
Use the shortcode `[fbcc_beef_chart]` (or legacy `[beef_chart]`) in any post or page, or use the Gutenberg block "French Beef Cuts Chart" in the block editor.

= Can I customize the appearance? =
Yes, you can modify the shortcode parameters:
- `[fbcc_beef_chart width="800px" height="500px"]` for custom dimensions
- `[fbcc_beef_chart width="100%" height="600px"]` for responsive width

= Is it secure? =
Yes, the plugin uses prepared statements, nonce verification, and follows WordPress security best practices.

= What beef cuts are included? =
The plugin includes 29 traditional French beef cuts with authentic names including Filet, Entrecôte, Rumsteck, Faux-filet, Côtes, and many more.

= Can editors manage the plugin? =
Yes, both administrators and editors can access and manage the beef chart settings.

= Is it mobile-friendly? =
Yes, the plugin is fully responsive and works perfectly on all devices.

= Does it support multiple languages? =
Yes, the plugin is internationalized and ready for translation.
